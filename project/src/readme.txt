To compile, either run:
make 

or

gcc -g -o client gameClient.c
gcc -g -o client gameServer.c

To run:

for server:
./server 'Ip address' 'port number'

for client:
./client 'port number'


Port number needs to be the same.
To run on the same computer, type 'localhost' instead of the IP address.

Valable inputs: s, p or r
Quit option: q
